OpenVM Opcodes and Hexadecimal Representation

Opcode Hexadecimal Representation Operation Description
MOV 0x01 [reg] [src] [value...] Move a value into a register. This works for both constants and variables. Example: MOV REG_A, VALUE will store a constant or variable value into register REG_A.
ADD 0x02 [reg] [value] Add an immediate value to the contents of a register. This works for both constants and variables. Example: ADD REG_A, 5 will add 5 to the value in REG_A, which can be a constant or variable.
SUB 0x03 [reg] [value] Subtract an immediate value from the contents of a register. This works for both constants and variables. Example: SUB REG_A, 3 will subtract 3 from the value in REG_A.
MUL 0x04 [reg] [value] Multiply the contents of a register by an immediate value. This works for both constants and variables. Example: MUL REG_A, 2 will multiply REG_A by 2.
DIV 0x05 [reg] [value] Divide the contents of a register by an immediate value. This works for both constants and variables. Example: DIV REG_A, 4 will divide REG_A by 4.
SQRT 0x06 [reg] Take the square root of the value in a register. This works for both constants and variables. Example: SQRT REG_A will replace REG_A with the square root of its current value.
PWR 0x07 [reg] [value] Raise the value in a register to an immediate power. This works for both constants and variables. Example: PWR REG_A, 3 will cube the value in REG_A.
PRINT 0x08 [string address] Print a string to the output. This works with string constants and string variables. Example: PRINT "Hello, World!" will print the message.
INPUT 0x09 [reg] Get input from the user and store it in a register. This works for variables. Example: INPUT REG_C will store user input in REG_C.
THREAD_SPIN_UP 0x0A [thread ID] Start a new thread with the specified ID. This works regardless of whether the thread ID is a constant or variable. Example: THREAD_SPIN_UP 2 starts a new thread with ID 2.
THREAD_TERMINATE 0x0B [thread ID] Terminate the specified thread. This works regardless of whether the thread ID is a constant or variable. Example: THREAD_TERMINATE 2 will stop thread with ID 2.
QUEUE_INIT 0x0C [reg] [queue] Initialize a queue for a specific register. Example: QUEUE_INIT REG_VAR_HEAP_PTR queue initializes a queue in a register.
PUSH_QUEUE 0x0D [queue] [value] Push a value onto the global queue. This works for both constants and variables. Example: PUSH_QUEUE queue 100 adds the value 100 to the global queue.
POP_QUEUE 0x0E [queue] [dest_reg] Pop a value from the global queue and store it in a destination register. This works for both constants and variables. Example: POP_QUEUE queue REG_A will store the first item in the queue into REG_A.
FUNCTION_CALL 0x0F [function ID] [arg1] [arg2] ... Call a function with the provided arguments. This works for both constants and variables. Example: FUNCTION_CALL print REG_C calls the print function with REG_C as an argument.
REGISTER_CONST_STRING 0x10 [string name] [value] Define a constant string in memory. Example: REGISTER_CONST_STRING GREETING "Hello from OpenVM!" defines a string constant.
REGISTER_VAR_INT 0x11 [var name] [value] Define an integer variable. Example: REGISTER_VAR_INT counter 0 defines an integer variable named counter with an initial value of 0.
REGISTER_VAR_FLOAT 0x12 [var name] [value] Define a floating point variable. Example: REGISTER_VAR_FLOAT counter 0.5 defines a float variable.
REGISTER_VAR_STRING 0x13 [var name] [value] Define a string variable. Example: REGISTER_VAR_STRING greeting "Hello!" defines a string variable.
